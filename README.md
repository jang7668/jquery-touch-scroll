jquery-touch-scroll
===================

HOW TO USE
----------

      $(element).touchScrollEvent({
          touch : true,
          swipeRight : function() { }, // Swipe Right Event
          swipeLeft: function() {}, // Swipe Left Event
          swipeUp : function() {}, // Swipe Up Event
          swipeDown : function() {}, // Swipe Down Event
          mouseWheel : true,
          scrollUp : function() {},
          scrollDown : function() {},
          touchThreshold : 5
       });
